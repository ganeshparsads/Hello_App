# Hello_App
Full stack application involving Flask &amp; React.

## Pre-requisites
Run the following commands in the project root directory before starting the application for the first time.
```bash
npm install --dev
```

## Usage
Application can be started with:

Bundle JSX files using webpack:

```bash
cd fullstack_template/static
npm run build
```

Start the python server:
```bash
cd fullstack_template/server
python build.py
```
